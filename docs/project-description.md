# Arcos — Description du projet

## Vision
Arcos est une plateforme de supervision industrielle légère, conçue comme alternative aux SCADA traditionnels. Elle permet aux PME industrielles de connecter leurs équipements, visualiser leurs données en temps réel et exploiter l'IA pour l'analyse et les alertes.

## Objectifs
- Démocratiser la supervision industrielle pour les PME
- Remplacer l'export manuel de données par du temps réel
- Offrir un déploiement en jours (pas en mois)
- Intégrer l'IA nativement pour l'analyse des données capteurs

## Cibles
- PME industrielles (automates, capteurs, PLCs)
- Bureaux techniques et équipes maintenance
- Industrie de l'eau, énergie, agroalimentaire, process

## Positionnement
> "L'alternative légère aux SCADA traditionnels — déployée en jours, pas en mois."
